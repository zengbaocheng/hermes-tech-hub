## 解析正则

```python
import re, json, sys
# curl返回直接cat进来
data = sys.stdin.read()
m = re.search(r'jQuery\((.+)\)', data)
d = json.loads(m.group(1))
items = d['data']['diff']
```

## Pitfalls

- **缺少 Referer header**: Eastmoney会返回403，确保加 `-H "Referer: https://quote.eastmoney.com"`
- **不翻页**: 概念板块总共486个，只拿pn=1只有100个，漏掉大量数据
- **sina HQ API**: `hq.sinajs.cn` 已废弃（返回Forbidden），不要用
- **腾讯行情API**: `qt.gtimg.cn` 返回 `v_pv_none_match`，不可用
- **JSONP解析**: 返回格式是 `jQuery({...})` 不是纯JSON，直接 `json.loads` 会报错

## 行业 vs 概念板块区别

| 类型 | fs参数 | 数量 | 特点 |
|------|--------|------|------|
| 行业板块 | `m:90+t:2` | ~100个 | 按主营业务分类，相对稳定 |
| 概念板块 | `m:90+t:3` | ~486个 | 按题材概念，热炒股相关，波动大 |

## 格式化倾向

主上收到数据后无异议，说明当前格式（表格+emoji+涨跌箭头）基本合适。科创50领跌时用`🔴`大字标注，整体跌多涨少时加总评。控制每板块数量，重要板块（涨跌幅大、热门概念）优先。
