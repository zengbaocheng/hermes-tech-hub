# Hermes 自主学习技能

## 重要说明

**本技能让 Hermes 能够根据主人提供的网址链接，自主学习新技能、工具使用方法和适用场景，并将其转化为可用的技能文件。**

## 功能概述

1. **网址内容获取**：读取主人提供的学习网址内容
2. **技能结构分析**：分析内容中的技能定义、使用方法、参数说明
3. **技能文件生成**：生成标准格式的 SKILL.md 文件
4. **情景场景库**：建立适用场景判断逻辑，自动匹配合适技能
5. **学习记录汇报**：向主上汇报学习成果和使用建议

## 输入格式

主人可以通过以下方式触发学习：

```
学习技能: [网址链接]
或
根据 [网址链接] 学习 [技能名称]
或
自主学习: [网址链接]
```

生成的技能文件必须包含以下结构：

```markdown

version: 1.0.0
author: hermes-self-learner
metadata:
  tags: [相关标签]

# [技能名称]

## 功能概述

## 情景场景库

### 场景自动匹配规则

| 用户意图关键词 | 匹配技能类型 | 示例 |
|--------------|------------|------|
| 创建/生成/制作 | 内容生成技能 | "创建一份报告" |
| 分析/研究/调查 | 分析研究技能 | "分析市场趋势" |
| 编程/开发/代码 | 编程开发技能 | "写一个Python脚本" |
| 写作/编辑/润色 | 写作编辑技能 | "润色这篇文章" |
| 搜索/查找/查询 | 信息检索技能 | "查找相关信息" |
| 总结/归纳/概括 | 总结归纳技能 | "总结这段文字" |
| 翻译/转换/改写 | 翻译转换技能 | "翻译成英文" |
| 计划/安排/调度 | 计划安排技能 | "安排下周计划" |

### 情景判断优先级

1. **精确匹配**：用户明确提到技能名称
2. **意图识别**：根据动词判断用户意图
3. **模糊匹配**：根据上下文推断需求
4. **建议推荐**：主动推荐可能需要的技能

### 主动推荐规则

当用户描述需求但没有匹配技能时：

1. **分析需求关键词**
2. **搜索相关学习资源**（如主人有提供）
3. **生成学习建议**
4. **推荐可能合适的通用技能**

## 使用限制

1. **版权注意**：学习内容仅供个人使用，不能直接复制商业内容
2. **信息验证**：对于关键信息需向主上确认
3. **权限限制**：学习前需确认主上授权
4. **质量保证**：生成技能后需主上审核

```

## 注意事项

1. **网址格式**：支持 http/https 链接
2. **内容要求**：最好是有结构化内容的网页
3. **学习时间**：复杂技能可能需要多次对话
4. **存储位置**：生成的技能默认放在 ~/.hermes/skills/autonomous-ai-agents/[技能名]/
